# OneWorld SatNet Landing Page

## Overview
This project is a landing page for OneWorld SatNet, a global satellite internet service provider. The website is designed to showcase the services, plans, partners, and provide a signup option for users.

## Project Structure
The project consists of the following files and directories:

- **index.html**: The main landing page featuring sections for the hero banner, services, coverage, partners, FAQs, testimonials, and contact information.
- **plans.html**: A detailed outline of the different plans offered, highlighting modified pricing at half the original rates, along with descriptions and features for each plan.
- **partners.html**: A page dedicated to listing the partners associated with OneWorld SatNet, including logos and descriptions of each partner organization.
- **signup.html**: A signup form for users to register for the service, including fields for user information and a submission button.
- **css/style.css**: Contains styles for the website, including layout, typography, colors, and responsive design elements to ensure compatibility across various devices.
- **js/main.js**: Includes JavaScript functionality for interactive elements, form validation, and dynamic content updates.
- **images/**: A directory for high-quality images related to various sections and plans, enhancing the overall design of the site.
- **README.md**: This documentation file, providing setup instructions, features, and relevant information for developers or users.

## Features
- Responsive design that adapts to different screen sizes.
- Interactive elements for a better user experience.
- Clear presentation of services and plans with attractive visuals.
- Easy navigation between different sections of the site.
- Signup functionality for user registration.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the project directory in your preferred code editor.
3. Open `index.html` in a web browser to view the landing page.
4. Modify the `css/style.css` and `js/main.js` files as needed to customize styles and functionality.
5. Add images to the `images/` directory to enhance the visual appeal of the site.
